# Model Monitoring

Production ML systems should monitor data drift, prediction drift, model quality, latency, throughput, and system errors.

Data drift can be detected using statistical tests such as Kolmogorov-Smirnov test, population stability index, Jensen-Shannon divergence, and feature distribution comparison.

Model quality can be tracked using delayed labels, precision, recall, F1 score, ROC AUC, PR AUC, and calibration metrics.
